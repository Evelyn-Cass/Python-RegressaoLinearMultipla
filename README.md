# 📊 Python-RegressaoLinearMultipla

A Python project for performing multiple linear regression analysis, built with Jupyter Notebook. Demonstrates how to model relationships between multiple independent variables and a dependent variable.

## 📱 About The Project

Python-RegressaoLinearMultipla is a simple and practical implementation of multiple linear regression. The project uses a housing dataset to predict prices based on multiple features.

This project was created for study purposes and to demonstrate multiple regression analysis using Python.

## ✨ Features

- Multiple linear regression model implementation
- House price prediction example
